## GO-JEK 
### parking Lot Assignment

#### Depandencies:
1) JDK
2) Maven
3) Junit

###### To install all dependencies, compile and run tests:
``$ bin/setup``
###### To run program via a file input:
``./parking_lot.sh  <input_filepath>``
###### To run the program and launch the shell:
``$ bin/parking_lot``
